# Венеция · Велнес

Премиальный женский велнес-клуб — многостраничный сайт.

## Страницы

| Файл | Описание |
|------|----------|
| `index.html` | Техническая навигация (карта всех страниц) |
| `venecia-multipage.html` | Основной сайт (SPA, 9 страниц) |
| `venecia-nav.html` | Навигационная разводящая с SVG-иллюстрациями |

## Структура сайта

- `#page-home` — Главная страница
- `#page-directory` — Разводящая «Все разделы»
- `#page-fitness` — Фитнес
- `#page-spa` — СПА и велнес
- `#page-clinic` — Клиника красоты
- `#page-beauty` — Салон красоты
- `#page-team` — Команда и видеогид
- `#page-booking` — Бронирование (6 форм)
- `#page-account` — Личный кабинет (7 разделов)

## Навигация

```js
navigate('fitness')   // перейти на страницу фитнеса
navigate('spa')       // СПА
navigate('booking')   // бронирование
```
